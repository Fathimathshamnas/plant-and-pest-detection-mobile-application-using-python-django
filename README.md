# 🌿 Plant and Pest Detection Mobile Application

Project Overview
This project is a mobile application developed using Python Django to detect plant diseases and pests using image processing and machine learning techniques.

Features
- Detect plant diseases from images
- Pest identification system
- User-friendly mobile interface
- Fast and accurate predictions

Technologies Used
- Python
- Django
- Machine Learning
- OpenCV
